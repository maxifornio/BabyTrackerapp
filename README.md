🍼 Baby Tracker App / https://baby-trackerapp.vercel.app/

Aplicación frontend desarrollada en React para el seguimiento de actividades diarias de un bebé. Permite registrar eventos importantes y visualizar información clave para el cuidado diario.

🚀 Funcionalidades
📝 Registro de actividades

Podés registrar las siguientes acciones:

🍼 Biberón
🍽️ Comida
🚶 Paseo
💤 Sueño
🎮 Juego
🧷 Pañal
📅 Visualización de datos
Actividades del día actual
Historial de actividades pasadas
⏱️ Contador de próximo biberón
Se reinicia automáticamente al registrar un biberón
Intervalo fijo de 4 horas entre biberones
Permite saber cuánto falta para el próximo
📊 Informe diario

Resumen automático del día:

Cantidad de pañales cambiados
Cantidad de biberones dados
🧠 Objetivo

Brindar una herramienta simple e intuitiva para organizar la rutina del bebé, facilitando el seguimiento de tareas y ayudando a mantener un control claro de los cuidados diarios.

🛠️ Tecnologías utilizadas
⚛️ React
JavaScript (ES6+)
CSS / Styled Components (o el que uses)
LocalStorage (si guardás datos en el navegador)
